## 🚀 START HERE

### Ejecutar todo en 3 pasos:

**1. Setup (primera vez):**
```bash
npm run setup
```

**2. Backend (Terminal 1):**
```bash
npm run dev:backend
```

**3. Frontend (Terminal 2):**
```bash
npm run dev:frontend
```

**4. Abre:** http://localhost:3000
